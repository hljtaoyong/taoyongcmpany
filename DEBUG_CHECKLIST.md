# 网站功能调试清单

## 1. 清除浏览器缓存

### Windows:
```
Ctrl + Shift + Delete → 清除缓存
Ctrl + Shift + R → 硬性重新加载
```

### Mac:
```
Cmd + Shift + Delete → 清除缓存
Cmd + Shift + R → 硬性重新加载
```

## 2. 检查控制台错误

按 F12 打开开发者工具，查看 Console 标签是否有红色错误。

## 3. 检查网络请求

在开发者工具的 Network 标签中：
- 刷新页面
- 查找失败的请求 (红色)
- 检查 Supabase 请求是否成功

## 4. 测试各个功能

### 截图功能测试:
1. 访问 /todos 页面
2. 按 Alt+S
3. 应该出现半透明蒙层
4. 拖拽选择区域
5. 点击确认

### 人生计时器测试:
1. 访问 /todos 页面
2. 滚动到页面底部
3. 应该看到彩色进度条
4. 鼠标悬停显示生命统计

### Google 登录测试:
1. 点击登录按钮
2. 应该弹出 Google 登录窗口
3. 授权后成功登录

## 5. 对比本地环境

```bash
npm run dev
```

访问 http://localhost:5173，对比哪些功能在本地正常，但在生产环境不正常。

## 6. 检查 Vercel 部署日志

1. 访问 Vercel Dashboard
2. 选择你的项目
3. 查看 Deployments 标签
4. 点击最新部署查看日志
5. 检查是否有红色错误
